## 🚀 Tecnologias Utilizadas

Este projeto foi construído com foco em performance, design moderno e boas práticas:

-   **HTML5 Semântico**: Estrutura acessível e otimizada para SEO.
-   **CSS3 Moderno**:
    -   Variáveis CSS (Custom Properties) para fácil manutenção de temas.
    -   Flexbox e Grid Layout para responsividade.
    -   Animações e transições suaves.
    -   Efeitos de "Spotlight" e Glassmorphism.
-   **JavaScript (ES6+)**: Interatividade leve e eficiente (Menu Mobile, Animações de Scroll, Contadores).
-   **Vite**: Build tool de próxima geração para desenvolvimento rápido.
-   **Google Fonts**: Tipografia premium com *Outfit*, *Inter* e *Lexend*.
-   **Phosphor Icons**: Ícones vetoriais flexíveis e modernos.

## 🎨 Design System

O projeto utiliza um tema escuro "Premium" com acentos em azul vibrante:

-   **Cores Principais**: Dark Blue (`#020617`), Surface (`#0F172A`), Accent (`#3B82F6`).
-   **Tipografia**:
    -   Títulos: *Outfit*
    -   Corpo: *Inter*

## 🛠️ Como Executar Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

### Pré-requisitos

-   [Node.js](https://nodejs.org/) instalado (versão 14 ou superior recomendada).

### Instalação

1.  Clone o repositório:
    ```bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    ```
2.  Entre na pasta do projeto:
    ```bash
    cd richarddev
    ```
3.  Instale as dependências:
    ```bash
    npm install
    ```

### Desenvolvimento

Para iniciar o servidor de desenvolvimento local:

```bash
npm run dev
```

O projeto estará disponível em `http://localhost:5173` (ou outra porta indicada no terminal).

### Build para Produção

Para gerar a versão otimizada para produção:

```bash
npm run build
```

Os arquivos estáticos serão gerados na pasta `dist`.
